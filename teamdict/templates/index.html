manual installation

  1. Browse to this url:
    https://my.slack.com/apps/build/custom-integration
  2. Select 'Slash Commands'
  3. Follow these steps for the /dbmod command
    • i. Enter the command /dbmod into the text field
    • ii. In the URL field, enter:
      {{ url_for('modify', _external=True) }}
    • iii. In the Customize Name field, enter "Team Dict"
    • iv. (Optional) Check the "Show this command in the autocomplete list" box:
      Description: "Modify a dictionary table"
      Usage Hint: "<command> <table> [<argument>]"
    • v. Click Save Integration
  4. Repeat Steps 1 and 2 for the /lookup command
  5. Follow these steps for the /lookup command
    • i. Enter the command /lookup into the text field
    • ii. In the URL field, enter:
      {{ url_for('lookup', _external=True) }}
    • iii. In the Customize Name field, enter "Team Dict"
    • iv. (Optional) Check the "Show this command in the autocomplete list" box:
      Description: "Lookup data from a dictionary table"
      Usage Hint: "<key> [<table>]"
    • v. Click Save Integration
